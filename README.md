# DesignCV 🚀
## Générateur de CV Intelligent en Temps Réel

**DesignCV** est une application web légère et performante permettant de créer, personnaliser et télécharger des CVs au format PDF directement dans le navigateur. Simple, rapide et sans inscription requise.

### ✨ Fonctionnalités (Version MVP)
- **Édition en temps réel :** Modifiez le formulaire à gauche, voyez le résultat instantanément à droite.
- **Personnalisation Avancée :** 3 thèmes de design (Classique, Moderne, Élégant) + 5 couleurs prédéfinies.
- **Gestion Dynamique :** Ajoutez autant d'expériences, formations et projets que vous voulez.
- **Compétences Intelligentes :** Système de tags (badges) pour les compétences techniques et soft skills.
- **Export PDF :** Génération haute qualité directement via le navigateur.
- **Sauvegarde Locale :** Historique des CVs sauvegardé automatiquement (LocalStorage).
- **Optimisation :** Outil pour ajuster l'espacement du CV (Aéré, Compact, Équilibré).

---

### 🛠️ Technologies Utilisées
- **Frontend :** HTML5, CSS3, JavaScript (Vanilla ES6+)
- **Librairies :** `html2pdf.js` (pour l'export PDF)
- **Polices :** Google Fonts (Inter, Playfair Display)
- **Architecture :** Application statique, côté client uniquement (pas de base de données).

---

### 🚀 Installation & Utilisation

DesignCV est une application statique. Aucun serveur n'est nécessaire.

1. Téléchargez ou clonez le projet.
2. Ouvrez simplement le fichier `index.html` dans votre navigateur (Chrome, Firefox, Edge recommandés).
3. Remplissez vos informations et cliquez sur "Télécharger PDF" quand vous êtes satisfait du rendu.
